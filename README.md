# La French Tech Tokyo homepage

Official website developed by the community that will be hosted at https://lafrenchtechtokyo.jp

## Setup

```
npm install
```

## Commands

Start dev environment

```
npm run dev
```

Build prod environment

```
npm run build
```

Start prod environment

```
npm run start
```
